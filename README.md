## 项目概述
面向通信行业从业者的 Starlink 分层认知与技术拆解公众号系列。

## 系列结构
### 第一阶段（6篇）
1. （01）Starlink 到底是什么？它为什么重新定义了卫星通信
2. （02）从一张图看懂 Starlink 的整体系统：卫星、地面站、终端、互联网出口怎么连起来
3. （03）Starlink 和传统卫星通信、5G、地面宽带，到底有什么本质区别
4. （04）为什么 Starlink 能把延迟打下来？答案不只是因为低轨
5. （05）Starlink 用的是什么频段？Ku、Ka、E-band 各自意味着什么
6. （06）从通信链路看 Starlink：上行、下行、回传链路分别怎么工作
